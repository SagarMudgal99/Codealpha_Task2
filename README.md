# 📊 Unemployment Analysis with Python

This project analyzes unemployment rate data using Python to understand unemployment trends and the impact of the Covid-19 pandemic. The analysis includes data cleaning, exploration, visualization, and extracting meaningful insights that can support economic and social policy decisions.

## 🚀 Features

* Data cleaning and preprocessing.
* Exploratory Data Analysis (EDA).
* Visualization of unemployment trends.
* Analysis of Covid-19's impact on unemployment.
* Identification of seasonal and long-term patterns.
* Data-driven insights and conclusions.

## 🛠 Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## 📊 Dataset

The dataset contains unemployment rate information over time, enabling trend analysis and the study of economic changes during the Covid-19 period.

## 🎯 Objective

To analyze unemployment patterns, evaluate the effects of the Covid-19 pandemic, and generate insights that could help inform economic and social policies.
